# 🏦 Object-Oriented C++ Banking Ledger System

A clean, terminal-based financial ledger application written in modern C++. This project demonstrates proper object-oriented programming (OOP) principles, state encapsulation, persistent file handling (`fstream`), and robust user authentication workflows.

## 🚀 Features
- **Secure Registration & Login:** Enforces a 9-character password minimum and utilizes iterative security verification loops instead of risky recursive calls.
- **Persistent Storage:** Local file system synchronization saves user data structures securely to `account_data.txt` upon every successful transaction.
- **Financial Transaction Logic:** Complete state encapsulation handling simulated user cash balances, validation steps, withdrawals, and deposits.

## 🛠️ Tech Stack & Concepts
- **Language:** C++ (Compiled with C++17 standard flags)
- **Concepts:** Encapsulation, object persistence, file streams, layout buffers, clean input handling.
